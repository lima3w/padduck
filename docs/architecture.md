
# Architecture

## Overview
Monolithic API-first architecture.

## Components
- Backend (Go)
- Frontend (React)
- PostgreSQL DB

## Design Principles
- API-first
- Stateless backend
- Strong data integrity
- Simple deployment

## Backend Layers
- handlers: HTTP layer
- services: business logic
- repository: DB access

## Data Model (Initial)
- users
- sections
- subnets
- ip_addresses
